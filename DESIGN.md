# DayFlow Design System

## Visual Identity
Clean, minimal, calm productivity aesthetic. Focus on whitespace, subtle borders, and a high-contrast but soft color palette.

## Colors

### Light Mode
- **Page Background:** `#F5F5F3`
- **Surface/Card Background:** `#FFFFFF`
- **Primary Text:** `#1A1A1A`
- **Secondary Text:** `#6B6B6B`
- **Outline/Border:** `#E2E2E2` (0.5px)

### Dark Mode
- **Page Background:** `#0F0F0F`
- **Surface/Card Background:** `#1A1A1A`
- **Primary Text:** `#F5F5F5`
- **Secondary Text:** `#A0A0A0`
- **Outline/Border:** `#333333` (0.5px)

### Accents
- **Teal (Primary Action/Routine):** `#1D9E75`
- **Purple (Tasks):** `#534AB7`

### Category Palette
- **Work:** Soft blue background + dark blue text
- **Health:** Soft green background + dark green text
- **Personal:** Soft purple background + dark purple text

## Typography
- **Font:** Manrope (Clean Sans-Serif)
- **Weights:** 400 (Regular), 500 (Medium)
- **Case:** Sentence case used globally for a friendly, approachable tone.
- **Scale:**
  - **Heading:** 24px, Medium
  - **Subheading:** 18px, Medium
  - **Body:** 16px, Regular
  - **Caption/Muted:** 14px, Regular

## Spacing & Radius
- **Border Radius:** 
  - 12px: Cards
  - 8px: Inputs, Buttons, Section containers
  - 4px: Pills/Badges
- **Margins:** 16px - 24px standard container padding.

## Component Patterns

### Cards
- **Style:** 12px radius, 0.5px border, minimal to no shadow.
- **Timeline Cards:** Features a 3px solid left border accent (Teal for routines, Purple for tasks).

### Buttons
- **Primary:** Filled Teal, 8px radius, white text.
- **Secondary/Outlined:** 0.5px border, 8px radius, accent-colored text.
- **Height:** 36px standard.

### Inputs
- **Style:** 36px height, 8px radius, 0.5px border. Background matches surface color.

### Progress Bar
- **Style:** 6px height, Teal fill, gray track, fully rounded.

### Checkboxes
- **Routine:** Circular.
- **Task:** Square.

### Pills
- **Style:** Small, 4px border radius, soft background color with matching high-contrast text.
